# Solar Meter Tracker PWA

วิธีใช้งานบน iPhone แบบแอป:

1. อัปโหลดทั้งโฟลเดอร์นี้ขึ้น Netlify Drop หรือ GitHub Pages
2. เปิดลิงก์เว็บด้วย Safari บน iPhone
3. กดปุ่ม Share > Add to Home Screen
4. เปิดจากไอคอนบนหน้าจอ iPhone

หมายเหตุ: ข้อมูลถูกเก็บในเครื่องของ Browser/หน้าเว็บนั้น ๆ ด้วย localStorage ควรกด Backup JSON เป็นระยะ
